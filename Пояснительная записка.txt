3. Описание идеи  
Проект создан для упрощения аренды недвижимости. Основная цель — помочь людям быстро находить жилье и безопасно оформлять сделки.  

4. Описание реализации  
Основные модули:  
- Аутентификация — вход, регистрация, профиль  
- Объявления — добавление, редактирование, удаление  
- Чат — общение между пользователями  
- Поиск — фильтрация объявлений  

Ключевые классы:  
- User — данные пользователя  
- Property — информация об объявлениях  
- Message — сообщения в чате  
- Amenity — удобства (Wi-Fi, парковка и т. д.)  

База данных:  
- SQLite (для тестов), PostgreSQL (для работы)  
- Таблицы: users, properties, messages, amenities, property_images  

5. Используемые технологии  
- Бэкенд: Python, Flask, SQLAlchemy  
- Фронтенд: HTML, CSS, JavaScript, Bootstrap  
- Дополнительно:  
  - Flask-Login для авторизации  
  - Flask-SocketIO для чата  
  - Werkzeug для защиты паролей
